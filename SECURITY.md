Reporting a Vulnerability
Esta seção orienta como reportar uma vulnerabilidade.

Se você encontrar uma vulnerabilidade, siga os passos abaixo para reportá-la:

Envio do Relatório:

Inclua informações sobre a versão do projeto e passos para reproduzir a vulnerabilidade.
Acompanhamento:

Você receberá uma confirmação de recebimento dentro de 48 horas após o envio do seu relatório.
Manteremos você atualizado sobre o status do seu relatório a cada 5 dias úteis.
Após a Avaliação:

Se a vulnerabilidade for aceita, trabalharemos em uma correção e a lançaremos em uma atualização de segurança o mais rápido possível.
Se a vulnerabilidade for declinada, você receberá uma explicação detalhada dos motivos.
Confidencialidade:

Pedimos que você mantenha a vulnerabilidade confidencial até que uma correção seja lançada para garantir que os usuários não sejam afetados.
Agradecemos sua contribuição para manter nosso projeto seguro e protegido.
